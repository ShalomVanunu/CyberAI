 
#### 2. התקנת ספריות

1. להנחות את הארכיטקט ליצור קובץ `requirements.txt`
2. להוסיף אליו את כל הספריות הדרושות
3. להתקין עם פקודת ההתקנה:
```
pip install -r requirements.txt
```
